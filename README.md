# AiBidV3

企业版 AI 投标书协作平台（Web 端）。

产品目标是建立从招标文件导入、智能解析、要求确认、响应矩阵、目录规划、协同写作、审核校验到投标书导出的完整闭环，并确保 AI 生成内容可追溯、关键要求不漏项、企业资料受权限控制。

## 产品主流程

```text
招标文件导入
    ↓
智能解析与人工确认
    ↓
要求 / 评分 / 资质响应矩阵
    ↓
投标书目录生成与要求映射
    ↓
多人分工与 AI 辅助写作
    ↓
合规检查、评审与定稿
    ↓
DOCX 正式导出
```
